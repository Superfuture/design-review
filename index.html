What it does

Reviews like a design lead would.

🎯

Ranked, not a dump

Findings sorted blocking → important → polish, so you fix what matters first.

📐

Real thresholds

WCAG contrast ratios, 45–75ch measure, motion timing, 44px tap targets — numbers, not opinions.

🖼️

URL, image, or code

Critique a live page, a screenshot, or a component file with line-level references.

⚡

Auto-apply fixes

Run with --apply and it edits the safe ones — contrast, focus states, alt text, reduced-motion.

♿

Accessibility built in

Semantics, focus order, labels, contrast, and reduced-motion checked every pass.

✨

Names the strengths

Ends with what's working and the single highest-leverage change to make first.

Install

Up and running in 30 seconds.

1

Add the marketplace and install the plugin:

# in Claude Code
/plugin marketplace add Superfuture/design-review
/plugin install design-review@superfuture

# …or drop the folder in manually
git clone https://github.com/Superfuture/design-review /tmp/dr && cp -r /tmp/dr/design-review/skills/design-review ~/.claude/skills/design-review
2

Then just ask: "design review officialjp.com" or "critique this paywall screenshot."
